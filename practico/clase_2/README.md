# Clase 2: Algunos métodos de clasificación

En esta clase se presenta el método exploratorio multivariado Análisis de Componentes Principales (PCA) y algunos métodos de clasificación más utilizados como el Análisis Discriminante Lineal (LDA), el Análisis Discriminante Cuadrático (QDA) y el método de clasificación de Bayes (Naive Bayes).

## Contenido

- *E4-AA2021_PCA_y_clasificación.ipynb*: notebook que introduce a PCA y a los métodos de clasificación mencionados.
- *Entrega2-vinos.ipynb*: Ejercicio para poner en práctica lo aprendido durante la segunda clase utilizando la base de datos de vinos propuesta por Fisher.
